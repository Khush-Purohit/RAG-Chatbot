To use this project:

1. Create a virtual environment and activate it:
python -m venv venv
.\venv\Scripts\activate

2. Install all the dependencies:
pip install -r requirements.txt

3. Go to backend directory and start the backend server:
cd backend
python main.py

4. Now create a new terminal and activate venv in it, and then start the frontend:
.\venv\Scripts\activate
cd frontend
streamlit run app.py

5. Now the project is running on the port 8501
